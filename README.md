<h1>ForoHub  </h1>
<h2>API Rest Java Spring Boot 3, Spring Security JWT y MySQL</h2>
<p>Desafío Alura para completar el curso de Java Spring Boot. Modela el backend de un foro de ayuda. Incluye modelos
de Tópicos por Curso y Respuestas. Posee un modelo de registro y autenticación de usuarios, y protege 
los endpoints usando JWT.</p>
<p>Usa MySQL como base de datos</p>
<p>Documentación con Springdoc OpenAPI</p>
<h2>Pasos a seguir:</h2>
<ul>
<li>Clonar el repositorio</li>
<li>Configurar el archivo application.properties con los datos de su base de datos</li>
<li>Correr el proyecto</li>
<li>Acceder a la documentación en http://localhost:8080/swagger-ui.html</li>
</ul>
<h2>Modelos</h2>
<p>El proyecto cuenta con los siguientes modelos:</p>
<ul>
<li>Usuario</li>
<li>Curso</li>
<li>Topico</li>
<li>Respuesta</li>
</ul>
